# 家庭幻兽奖励 H5 产品化路线图

## 当前基线

- Vue 3 + TypeScript + Vite + Pinia + IndexedDB + PWA。
- 单孩子、单设备、本地优先。
- 孩子任务提交、家长审核、积分成长、奖励兑换和备份恢复闭环可用。
- Vue H5/PWA 是仓库内唯一产品实现，旧 Flutter/Android 工程已移除。

## 产品原则

- 鼓励优先，不使用惩罚、死亡、离开或焦虑化表达。
- 成长值与现实奖励积分分离。
- 家长操作必须可追溯，孩子端反馈必须清楚且温和。
- 先保证单家庭日常可用，再增加账号、云同步和复杂养成。
