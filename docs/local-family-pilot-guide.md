# Family Pets 本地家庭试用说明

## 开始使用

1. 在手机浏览器打开应用，填写孩子昵称、宠物名称和新的 4 位家长 PIN。
2. 首次设置不要求逐项配置任务或奖励；家长可稍后在家长中心调整推荐模板。
3. 浏览器出现“添加到主屏幕”时可以安装，也可以继续直接使用网页版本。

## PIN 的用途

- PIN 只用于同一台设备上的孩子端与家长端区分，不是账号密码。
- 忘记 PIN 时可从输入页选择“重新设置”，无需旧 PIN，也不会清空家庭数据。
- 修改或重设 PIN 后，家长中心会锁定，需要使用新 PIN 重新进入。

## 本地数据与备份

- 孩子资料、任务、积分和兑换记录只保存在当前浏览器，不会自动上传。
- 导出的 JSON 备份包含家庭业务数据，但不包含家长 PIN。
- 建议首次使用一周后导出一次，此后至少每月导出一次，并保存在可信的位置。
- 恢复前会显示孩子、等级、积分和记录数量；恢复会保留当前设备的 PIN。
- 不要把原始备份文件发送给试用反馈收集者。

## 离线与更新

- 完成首次加载后，核心页面、宠物图片和本地操作可离线使用。
- 有新版本时应用会主动提示，不会静默刷新正在填写的表单。

## 试用反馈清单

每周只记录以下简短结果，不需要上传原始家庭数据：

- 首次配置大约用了几分钟，哪一步最不清楚？
- 本周孩子主动打开应用几天？
- 家长每天确认任务大约需要多久？二次确认是否过多或过少？
- 孩子是否能在第一天完成一次小奖励闭环？
- 常规奖励和大奖励的等待时间是否合适？
- 已批准的奖励是否容易忘记兑现？
- 是否遇到离线、安装、更新或备份恢复问题？
- 是否强烈需要家长与孩子使用不同设备？请描述使用场景，不收集儿童身份信息。
